# Contributing to skret

## Development Setup

1. Install [mise](https://mise.jdx.dev/)
2. Run `mise run setup`
3. Make your changes
4. Run `mise run lint` and `mise run test`
5. Commit with `feat:` or `fix:` prefix

## Commit Convention

Only two prefixes allowed:
- `feat:` — new features
- `fix:` — bug fixes

## Pull Requests

- One PR per feature/fix
- Tests required (>=95% coverage)
- All CI checks must pass

# skret

Cloud-provider secret manager CLI wrapper with Doppler/Infisical-grade DX.

> **Status:** In development. v0.1.0 coming soon.

## Features (v0.1)

- Wrap AWS SSM Parameter Store with ergonomic CLI
- `skret run --` replaces `doppler run --`
- Multi-environment config (`.skret.yaml`)
- Import from Doppler, Infisical, dotenv
- Sync to GitHub Actions secrets
- Local YAML provider for development
